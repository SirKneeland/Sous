
# Milestones

## Milestone 1: Canvas UX (No AI)
- Render recipe from local JSON
- Steps are checkable
- Done steps lock visually
- Current step focus
- State persists via localStorage
- Demo route shows cooking mode

## Milestone 2: LLM + Patching
- Chat input sends recipe state to LLM
- LLM returns structured patches
- Patches apply cleanly
- Done steps are immutable
- Undo last patch

## Milestone 3: Cooking Polish
- Single-step cooking mode
- Swipe / tap to advance
- Optional timers
- Visual diff animation on patches
